# 相对原附件的主要修改

1. 锁定 `numpy<2`、兼容 SciPy 和 PYPOWER 版本，解决按原依赖安装后无法导入的问题。
2. 将“求解器数值收敛”与“工程可行”分开判断。
3. 只有电压、线路和 Q 限额均合格时才生成 `final_feasible_case.py`。
4. 失败结果改名为 `last_converged_but_infeasible_case.py` 或 `last_modified_case.py`。
5. 修复最大尝试次数用尽后仍记录未经验证动作的问题。
6. 负荷方案均从同一基准算例计算，不再累计乘法。
7. 增加初始牛顿雅可比矩阵秩、条件数、最小奇异值和奇异性判断。
8. 增强 PV/REF 无功越限和 PV→PQ 处理，考虑同一母线多台机组。
9. 向 LLM 提供原始矩阵证据、规则问题、求解尝试和数值诊断。
10. 增加 IEEE 9 节点、可修复坏数据和 Q 限额演示算例。
11. 将自动测试扩展到 6 项，包括成功状态、最终文件、负荷基准和尝试次数。
12. 增加 VS Code tasks、F5 调试配置、完整演示脚本和中文答辩说明。
